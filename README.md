## Описание программы

Это GUI-приложение для управления списком прочитанных книг. Позволяет добавлять книги, фильтровать по жанру и количеству страниц, сохранять и загружать данные в JSON-файл.

## Инструкция по запуску

1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте или скклонируйте репозиторий.
3. Запустите файл `book_tracker.py` командой:

```bash
python book_tracker.py
